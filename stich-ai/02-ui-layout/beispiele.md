# UI Layout Beispiele

---

## Beispiel 1: SaaS Dashboard

- Sidebar links (Navigation)
- Topbar (Search + Profile)
- KPI Cards oben
- Chart Section in der Mitte
- Data Table unten

---

## Beispiel 2: Finance App

- Balance Card oben
- Transaction List darunter
- Chart rechts
- Quick Actions Buttons

---

## Beispiel 3: Admin Panel

- Sidebar Navigation
- User Table als Hauptfokus
- Filters oben
- Detail Panel rechts

---

## Beispiel 4: Mobile App

- Bottom Navigation
- Cards stacked vertical
- Swipe Sections
- Floating Action Button

---

## Beispiel 5: Analytics Tool

- Full-width Charts
- KPI Grid oben
- Sidebar für Filter
