# Häufige UI Layout Fehler

---

## 1. Kein klares Layout

Problem:
Alles ist irgendwo platziert

Lösung:
Immer Sidebar + Content Struktur nutzen

---

## 2. Zu viele Elemente

Problem:
UI ist überladen

Lösung:
Nur 1 Hauptfokus pro Screen

---

## 3. Keine Hierarchie

Problem:
Alles wirkt gleich wichtig

Lösung:
- Headline > Cards > Details

---

## 4. Mobile vergessen

Problem:
Desktop Layout wird einfach verkleinert

Lösung:
Mobile = komplett eigenes Layout

---

## 5. Keine Struktur im Grid

Problem:
Chaos Positionierung

Lösung:
8px / 12 Column System nutzen

---

## 6. Keine klare Navigation

Problem:
User weiß nicht wohin

Lösung:
Sidebar oder Bottom Nav immer definieren
