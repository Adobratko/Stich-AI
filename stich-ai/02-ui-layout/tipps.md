# UI Layout Tipps für bessere Ergebnisse

---

## 1. Immer Layout-Typ definieren

Ohne Layout-Typ wird das Design generisch.

Beispiele:
- SaaS Dashboard
- Analytics App
- Admin Panel
- Finance Tracker

---

## 2. Daten zuerst denken

Stich AI funktioniert besser wenn klar ist:

👉 Welche Daten werden gezeigt?

Beispiele:
- Umsatz
- Nutzer
- Charts
- Tabellen

---

## 3. Sidebar ist fast immer Pflicht

Für Web Apps:

- Navigation links
- Content rechts

---

## 4. Mobile ist KEIN Desktop

Mobile Layout =

- Stack statt Grid
- weniger Inhalte
- größere Buttons

---

## 5. Weniger ist mehr

Gute Layouts haben:

- klare Struktur
- wenig Chaos
- Fokus auf 1 Hauptbereich

---

## 6. Komponenten vorher denken

Bevor du Layout generierst:

- Welche Karten?
- Welche Daten?
- Welche Actions?

---

## PRO TIP

Je klarer du Struktur gibst, desto „Apple-Level“ wird das UI.
