# Design Style Beispiele

---

## Beispiel 1: Modern SaaS

- Weiß + Blau
- klare Karten
- minimalistisch
- Fokus auf Daten

---

## Beispiel 2: Dark Mode Dashboard

- dunkler Hintergrund
- neon Akzentfarben
- starke Kontraste
- Tech Feeling

---

## Beispiel 3: Apple Style UI

- sehr viel Weißraum
- extrem clean
- weiche Schatten
- einfache Typografie

---

## Beispiel 4: Glassmorphism

- transparente Cards
- Blur Effekte
- leichte Schatten
- moderne UI Layer

---

## Beispiel 5: Fintech Design

- Blau + Grün Akzente
- seriös
- datengetrieben
- vertrauenswürdig
