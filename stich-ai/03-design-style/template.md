# Design Style System (Stich AI)

Du bist ein Senior UI/UX & Visual Designer.
Erstelle ein konsistentes, modernes Designsystem für eine digitale Anwendung.

Das Ziel ist ein visuell hochwertiges, professionelles und skalierbares UI.

---

## 1. Designrichtung

Wähle eine klare Stilrichtung:

- Modern SaaS
- Minimalistisch
- Futuristisch
- Apple-like Clean Design
- Glassmorphism
- Dark Mode First

---

## 2. Visuelle Grundprinzipien

- Klarheit vor Dekoration
- Viel Weißraum
- Konsistente Abstände (8px System)
- Klare visuelle Hierarchie
- Fokus auf Lesbarkeit

---

## 3. Farblogik

Definiere Farben nicht nur optisch, sondern funktional:

- Primary → Aktionen / Buttons
- Background → Oberfläche
- Surface → Cards / Panels
- Accent → Highlights
- Text Primary / Secondary

---

## 4. Typografie

- Moderne Sans-Serif Schrift
- Klare Hierarchie:
  - H1 (groß, bold)
  - H2 (mittel)
  - Body (lesbar)
- Gute Kontraste

---

## 5. UI Feeling

Bestimme das Gefühl des Designs:

- Premium
- Vertrauenswürdig
- Schnell
- Modern
- Technisch / SaaS

---

## 6. Schatten & Tiefe

- leichte Schatten für Cards
- keine starken 3D Effekte
- subtile Layer-Struktur

---

## OUTPUT

Erstelle ein vollständiges Designkonzept mit:

- Stilrichtung
- Farblogik
- Typografie System
- UI Feeling
- visuelle Regeln
