# Color System (Stich AI)

Du bist ein Senior UI Designer und Design System Architect.

Erstelle ein konsistentes, funktionales und skalierbares Farb-System für eine moderne digitale Anwendung.

---

## 1. Farbstrategie

Farben sind nicht dekorativ, sondern funktional.

Jede Farbe hat eine klare Bedeutung im UI.

---

## 2. Core Colors

Definiere die Hauptfarben:

- Primary Color (Hauptaktionen / Marke)
- Secondary Color (Unterstützung)
- Accent Color (Highlights / Fokus)
- Background Color (Gesamte Fläche)
- Surface Color (Cards / Panels)
- Text Primary / Secondary

---

## 3. Semantic Colors (WICHTIG)

Farben für Bedeutungen:

- Success (Erfolg)
- Warning (Warnung)
- Error (Fehler)
- Info (Information)

---

## 4. Farbrollen im UI

Definiere exakt wo Farben verwendet werden:

- Buttons
- Links
- Cards
- Charts
- Alerts
- Navigation

---

## 5. Dark & Light Mode

Das System muss funktionieren in:

- Light Mode
- Dark Mode

Regeln:
- gleiche Struktur
- nur invertierte Werte
- keine neuen Farben hinzufügen

---

## 6. Kontrast & Lesbarkeit

- hoher Kontrast für Text
- klare Lesbarkeit
- keine zu ähnlichen Farben
- Accessibility beachten

---

## 7. UI Stimmung

Farben definieren Emotion:

- Vertrauen (Blau)
- Energie (Rot/Orange)
- Wachstum (Grün)
- Innovation (Lila)

---

## OUTPUT

Erstelle ein vollständiges Color System mit:

- HEX Farbwerten
- Bedeutung jeder Farbe
- UI Verwendung
- Light & Dark Mode Struktur
